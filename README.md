日本传统色Lite
---
与人工作室出品，采用 NIPPON COLORS 精选的 250 个日本古典文学中的代表颜色，为设计师提供色彩灵感。

使用了 [WeUI for 小程序](https://github.com/Tencent/weui-wxss) UI 组件库
